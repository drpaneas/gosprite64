# Grid Utilities

The `math2d` package provides generic grid data structures for spatial operations. Import it with:

```go
import "github.com/drpaneas/gosprite64/math2d"
```

Covers `math2d.Grid[T]`, `Run`, `GridCell`, `ScanRow`, and `ScanCol`.
